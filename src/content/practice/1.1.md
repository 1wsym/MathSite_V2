# Действительные числа
## Целые и рациональные числа

---

<section 
  class="interactive-task" 
  question="Представьте бесконечную периодическую дробь $0,(3)$ в виде обыкновенной дроби." 
  correctAnswer="1/3" 
  solution="Пусть $x = 0,333...$, тогда $10x = 3,333...$. Вычитаем первое из второго: $9x = 3$, откуда $x = \frac{3}{9} = \frac{1}{3}$.">
</section>

<section 
  class="interactive-task" 
  question="Вычислите значение выражения: $|-15,5| + |5,5|$" 
  correctAnswer="21" 
  solution="Используем определение модуля: $|-15,5| = 15,5$, а $|5,5| = 5,5$. Сумма: $15,5 + 5,5 = 21$.">
</section>

<section 
  class="interactive-task" 
  question="Найдите сумму бесконечной геометрической прогрессии, если $b_1 = 12$ и $q = \frac{1}{3}$." 
  correctAnswer="18" 
  solution="По формуле суммы бесконечно убывающей геометрической прогрессии $S = \frac{b_1}{1-q}$: $S = \frac{12}{1 - 1/3} = \frac{12}{2/3} = \frac{12 \cdot 3}{2} = 18$.">
</section>

<section 
  class="interactive-task" 
  question="Укажите, каким числом (рациональным или иррациональным) является сумма $\sqrt{2} + (2 - \sqrt{2})$." 
  correctAnswer="рациональным" 
  solution="Раскрываем скобки: $\sqrt{2} + 2 - \sqrt{2} = 2$. Число $2$ можно представить в виде дроби $\frac{2}{1}$, следовательно, оно является рациональным.">
</section>

<section 
  class="interactive-task" 
  question="Найдите значение $x$, если $|x - 3| = 0$." 
  correctAnswer="3" 
  solution="Модуль числа равен нулю только в том случае, если само число равно нулю. Значит, $x - 3 = 0$, откуда $x = 3$.">
</section>
